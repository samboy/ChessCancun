These are multi-color (where the colors are black and white) version of the
icons, so we can make a compact COLR font which works with any modern
browser (even Safari) as well as Libre Office.

If nanoemoji is installed, the scrip make.nanoemoji.sh will make a font,
build/Font.ttf, which is a COLR version of Chess Cancun (ChessCancunColor).
