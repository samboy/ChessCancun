There are two fonts here:

* ChessCancun, for legacy systems which do not support color fonts.
  This version of the font uses a diagonal lattice to indicae dark
  squares, the way Chess books in the 19th and 20th century did.
  Details are in the file `ChessCancun.md`.
* ChessCancunColor, for modern setups (all modern browsers, including
  Safari, as well as LibreOffice) with support for color fonts.
  While the only colors the font has are black and white, this version 
  of the font, while being more compact than ChessCancun, looks more
  attractive on systems using light grey for the dark squares (or,
  for that matter, using medium and medium-light grey for the squares
  in dark mode).  Details are in `ChessCancunColor.md`.

Note that the two fonts have different mappings for letters.

They are both versions of ChessMerida where:

* The font images were converted in to high resolution PNG images
* The PNG images were converted in to SVG images using Inkscape
* The two fairy pieces from Capablanca Chess have been added

This makes the font public domain in a large number of jurisdictions, 
including the US.  See `COPYING.md` for details.
